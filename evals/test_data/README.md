# Test Data

This directory should contain sample files for testing the moderation agents:

## Images
- `professional_image.jpg` - A professional business image (e.g., product photo)
- `landscape.jpg` - A neutral landscape photo
- `product_photo.jpg` - Product photograph
- `id_card.jpg` - Sample ID card (for PII testing)
- `faces.jpg` - Image with faces (for PII testing)

## Audio
- `professional_greeting.wav` - Professional audio greeting
- `customer_inquiry.wav` - Customer service inquiry
- `contact_info.wav` - Audio with contact information

## Video
- `product_demo.mp4` - Product demonstration video
- `tutorial.mp4` - Tutorial video

Note: These files are not included in the repository. You should create or obtain 
appropriate test files for your evaluation needs.

For quick testing, you can use any appropriate sample files from royalty-free sources.
